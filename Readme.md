### Installation

You only need a C compiler.
>Tested using xcode


To run the tests you need to be sure that unit_testing is defined in main.h
```C
#define UNIT_TESTING
```
and to have in the working directory :
 * The CryptDecrypt executable
 * A tmp folder
```sh
$ mkdir tmp
```
 * The picture.jpg
